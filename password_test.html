Password Management Quiz

1. What is a password?

  • A) A way to store usernames safely
  • B) A combination of letters, numbers, and special characters used to protect accounts
  • C) A backup key for recovering lost data

2. Why are passwords important?

  • A) They make it easier to remember login details
  • B) They help keep accounts secure from unauthorised access
  • C) They speed up internet browsing

3. Which of the following is NOT a consequence of using a weak password?

  • A) Personal information being stolen
  • B) Your device running out of storage space
  • C) Unauthorised access to your online accounts

4. Which of these passwords is considered strong?

  • A) John123
  • B) Password1
  • C) B3!red@2

5. Which is NOT a quality of a strong password?

  • A) It contains a mix of uppercase and lowercase letters, numbers, and special characters
  • B) It is the same for all your accounts
  • C) It is at least 12-16 characters long

6. What is a good way to create a strong password?

  • A) Using a phrase with unrelated words and special characters
  • B) Using your name and birth year for easy recall
  • C) Keeping the same password for multiple accounts

7. What is Two-Factor Authentication (2FA)?

  • A) A second layer of security that requires an additional step beyond a password
  • B) A method of writing passwords down twice to make them stronger
  • C) A way to reset passwords if they are forgotten

8. Which of the following is NOT a type of 2FA?

  • A) SMS code sent to your phone
  • B) Using facial recognition or fingerprints
  • C) Re-entering the same password twice

9. When should you update your password?

  • A) Only when you create a new account
  • B) Every few months for important accounts or if you suspect it has been compromised
  • C) Never, as changing passwords can make them harder to remember

10. If you suspect your password has been compromised, what should you do first?

  • A) Continue using the same password to avoid confusion
  • B) Change the password immediately and update similar ones
  • C) Ignore it unless you receive an official warning
