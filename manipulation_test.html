Manipulation Techniques Used by Attackers Quiz

1. What is manipulation in cyber security?

  • A) A technique used to fix system vulnerabilities
  • B) The use of tactics to deceive individuals into revealing sensitive information
  • C) A method of encrypting passwords to improve security

2. Which of the following is an example of psychological manipulation?

  • A) An attacker creates a fake website to steal login credentials
  • B) A hacker exploits a software vulnerability to install malware
  • C) An attacker sends a fraudulent email that creates a sense of urgency to trick the victim

3. What is the main goal of pretexting?

  • A) To create a fake scenario to trick a victim into providing information
  • B) To use malicious software to steal personal data
  • C) To sell stolen credentials on the dark web

4. Which of the following best describes impersonation in cyber attacks?

  • A) A hacker disguising malware as a software update
  • B) An attacker pretending to be a trusted person to gain information
  • C) Using a fake Wi-Fi network to steal login credentials

5. Which tactic is commonly used in phishing attacks?

  • A) Sending urgent emails pretending to be from trusted companies
  • B) Encrypting personal data to protect users from cyber threats
  • C) Using biometric authentication to secure accounts

6. What is typosquatting?

  • A) Registering fake domain names that look similar to real ones
  • B) Using AI-generated deepfake videos to impersonate individuals
  • C) A technique where attackers guess passwords based on common words

7. Which of the following is an example of an attacker exploiting authority to manipulate a victim?

  • A) Offering free software downloads that contain malware
  • B) Posing as a government official and demanding immediate payment for unpaid taxes
  • C) Sending an email with a fake prize offer

8. What is a common sign of a homoglyph attack?

  • A) A fake website using letters that look similar, such as “g00gle.com” instead of “google.com”
  • B) An email containing an attachment that automatically downloads malware
  • C) A hacker creating a deepfake video of a CEO

9. Why is social media a valuable tool for cyber criminals?

  • A) Social media companies provide them with personal details
  • B) It helps them create stronger security systems
  • C) They can use public information to create highly targeted attacks

10. What is one way cyber criminals use AI and deepfakes in their attacks?

  • A) To create realistic voice clones for impersonation scams
  • B) To improve system security against cyber threats
  • C) To develop secure encryption algorithms
